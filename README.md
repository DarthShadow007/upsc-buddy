UPSC Buddy 🎓

An AI-powered, full-stack educational platform designed specifically for UPSC Civil Services Examination preparation. UPSC Buddy offers personalized mock tests, practice questions, flashcards, and current affairs tracking.

🚀 Features

- AI-Generated Mock Tests: Generate unique, UPSC-level GS Paper I and CSAT Paper II tests instantly using Gemini AI. No questions are ever repeated.
- AI Proctoring: Webcam-based proctoring during mock tests to simulate real exam environments.
- Smart Practice Bank: 200+ standard questions with dynamic filtering by subject and difficulty.
- Vocabulary & Flashcards: Tools to build and retain important terminology and concepts.
- Current Affairs Tracker: Stay updated with the latest news relevant to the civil services syllabus.
- Progress Dashboard: Track study time, mock test history, and overall performance metrics.
- Secure Authentication: User login and session management powered by Clerk.

🛠️ Tech Stack

Frontend (Vercel):
- React + Vite
- Tailwind CSS (Styling)
- Clerk (Authentication)

Backend (Render):
- Node.js + Express
- Prisma (ORM)
- PostgreSQL (Neon Database)
- Google Gemini API (AI content generation)

⚙️ Local Development Setup

 Prerequisites
- Node.js installed
- A PostgreSQL database (e.g., Neon)
- Clerk account for Auth keys
- Google Gemini API key
